TaskFlow ProSync Data Fields

This topic describes the data fields that are used in the TaskFlow ProSync application.

Project Information

The following data fields are related to project information:

Table 1. Project Information Fields
Field Name Description
Project ID Unique identifier for the project.
Project Name Name or title of the project.
Start Date Commencement date of the project.

Data records

The following data fields help with managing records in the Data Management module:

Table 2. Data Record Fields
Field Name Description
Record ID Identifier for the data record.
Data Type Type or category of the data.
Timestamp Date and time when the data was recorded.
